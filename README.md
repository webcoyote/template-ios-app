TemplateSwiftApp

This repository is an opinionated template for building an iOS app.

# Start Here

Rename the application:

```bash
./moveit-moveit "NewProjectName" "mydomain.com"
```

# Development Commands

One-time setup:

```bash
scripts/setup   # Install build tools and direnv
direnv allow    # Add scripts to the path and optional env vars in `./.env`
```

Useful commands:

```bash
build           # Build app
run             # Run the app
br              # Build and run the app
watch           # Build whenever changes occur
ide             # Open the project in xcode
screenshots     # Generate screenshots for AppStore
build-device    # Build for your phone
run-device      # Run the app on your phone
```
